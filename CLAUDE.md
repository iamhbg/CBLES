# CLAUDE.md — 한반도 화성암 ITS 개발 하네스

> 이 파일은 코드를 건드리는 모든 AI 에이전트 세션이 **작업 전에 먼저 읽는** 가드레일이다.
> 핵심 원칙: 이 프로젝트의 실패는 "빌드 깨짐"이 아니라 **전문가만 잡아내는 조용한 사실 회귀**다.
> 따라서 아래 §1~§3의 불변식은 임의로 바꾸지 말 것. 바꿔야 한다면 먼저 사용자에게 확인한다.
>
> `its_korea_igneous.py` 상단 docstring에도 이 문서를 가리키는 역참조가 있다 — 두 파일은
> 서로 연동되어 있으므로, 코드만 단독으로 열어도 이 하네스 문서로 돌아오게 되어 있다.

## 0. 프로젝트 개요

- **무엇**: 2022 개정 고등학교 「지구과학」 단원용 지능형 튜터링 시스템(ITS)
- **형태**: 단일 파일 Streamlit 앱(`its_korea_igneous.py`) — 좌측 시뮬레이터 + 우측 Gemini 챗봇
- **대상**: 고등학생 / **용도**: 수업 시연·교육 연구
- **설계 철학**: 구성주의 소크라테스식 튜터. 정답을 주입하지 않고 발문으로 학생이 개념을 **스스로 구성**하게 한다.
- **단원**: Ⅱ. 지구의 역사와 한반도의 암석
- **주제**: 한반도 화성암의 시대적·공간적 변천사 (마그마 조성 → 암석 종류 → 한반도 분포)

## 1. 절대 불변 규칙 — 교육과정 경계 (가장 중요)

- 작업은 성취기준 **[12지구02-03]** ("변동대에서 마그마가 생성되고, 그 조성에 따라 다양한 화성암이 생성됨을 설명할 수 있다") **범위 안에서만** 한다.
- **성취수준 A = ITS의 State 4**(최종 도달 상태)로 매핑되어 있다. 이 대응을 깨지 말 것.
- **결정적 지침** (교육과정 '성취기준 적용 시 고려 사항'):
  > "화성암의 **종류**보다는 화성암이 생성되는 **고유의 환경이 가지는 의미**를 이해하는 데 중점을 두며, **한반도에 나타나는 대표적인 지형과 연계**해서 수업을 전개한다."
  → 암석명 암기를 유도하는 방향으로 바꾸지 말 것. 환경의 의미 + 한반도 지형 연계가 축이다.
- **포함 금지**: **화산체 형태(순상화산/성층화산)**. 이 개념은 [12지시01-04] 지구시스템과학 소관으로 **이 단원 밖**이다. 시뮬레이터·시스템 프롬프트·출력 어디에도 다시 넣지 말 것. (구버전 ITS에 있던 항목이라 재유입 위험이 높음 — 항상 점검.)
- 2022 개정 교육과정을 벗어나는 심화(예: 절대연령 측정 원리의 상세 수식, 판구조론 정량 모델 등)는 추가하지 않는다.

## 2. 절대 불변 규칙 — 지질학 사실 (오류 시 즉시 회귀)

### 2-1. SiO₂ 함량 분류
- **≤ 52%** : 염기성(현무암질) — 어두운색
- **52 ~ 63%** : 중성(안산암질)
- **≥ 63%** : 산성(유문암질) — 밝은색
- 코드 상수: `SIO2_MAFIC_MAX = 52`, `SIO2_FELSIC_MIN = 63`. 경계값(52→현무암질, 63→유문암질) 동작을 바꾸지 말 것.

### 2-2. 암석 분류표 (교과서 표 II-1)
| 구분 | 산출/조직 | 염기성 | 중성 | 산성 |
|---|---|---|---|---|
| **화산암** | 지표 급랭·세립질 | 현무암 | 안산암 | 유문암 |
| **심성암** | 지하 서랭·조립질 | 반려암 | 섬록암 | 화강암 |

이 6칸 매핑은 사실이다. 어떤 칸도 임의로 바꾸지 말 것.

### 2-3. 마그마 생성 환경
- **발산 경계**(압력 감소) → 현무암질
- **섭입대**(물 공급 → 맨틀 부분 용융) → 현무암질
- **대륙 지각 가열·용융** → 유문암질
- **마그마 혼합 / 결정 분화** → 안산암질

### 2-4. 냉각 위치 (깊이 슬라이더 0~12km, 3단계)
- **지표**(깊이 0) : 화산암 · 세립질
- **천부**(0 < d < 3km) : 반심성암 · 반상 조직
- **심부**(≥ 3km) : 심성암 · 조립질

### 2-5. 대표 화성암 — 속성·지형·시대 매핑 (서로 뒤바꾸면 오류)
- **현무암** (교과서 대표 사례): **신생대** 화산활동, **지표 급랭**, **주상 절리**
  → 제주도(섭지코지·성산일출봉·중문대포 주상절리대), 한탄강(재인폭포), 울릉도·독도
- **화강암** (교과서 대표 사례): **중생대** 지각변동, **유문암질 마그마의 지하 서랭**, **판상 절리**, 융기·삭박으로 지표 노출
  → 설악산(비선대), 북한산(인수봉), 월출산(구정봉)
  → (배경 맥락: 쥐라기 대보 화강암, 백악기 불국사 화강암)

> ⚠️ 흔한 회귀 유형: 현무암↔중생대, 화강암↔지표 급랭, 제주도↔화강암 식으로 **속성을 교차 오염**시키는 것. 변경 시 반드시 위 표와 대조.

## 3. 절대 불변 규칙 — 교수설계 불변식

- **5단계 학습 상태기계(State 0~4)** 유지, 성취수준 E~A와 연계.
- **두 인과축** 유지:
  1. **물질적/조성 축**: 판 경계 환경 → 마그마 조성 → 암석 종류
  2. **시공간적/냉각분포 축**: 냉각 위치 → 조직(화산암/심성암) → 한반도 시대별·공간적 분포
- **출력당 소크라테스식 발문은 정확히 1개.** 한 번에 여러 개를 쏟아내지 않는다.
- **정답 선제 노출 금지.** 학생이 도달하기 전에 결론을 말해주지 않는다.
- **비평가적 어조.** "틀렸습니다" 류 표현 금지. 구성주의 촉진자 입장을 유지한다.
- **시뮬레이터 슬라이더 2개 고정**: SiO₂ 함량(45~75%), 냉각 깊이(0~12km).

## 4. 코딩 규약

- **코드 다이어트 금지**: 함수 생략·축약 금지. 모든 함수를 완전히 구현하고 예외 처리를 철저히 한다. (사용자 명시 요구)
- **단일 파일 아키텍처** 유지.
- **우아한 degradation**: `GENAI_AVAILABLE` 플래그로 라이브러리 누락 시에도 앱이 죽지 않게.
- **API 키는 학생이 입력하지 않는다.** 사이드바의 "Gemini API 키" 입력 칸은 **제거됨**(의도된 변경 — 학생용 배포 앱이라 개인 키를 받을 필요가 없음). `resolve_api_key()`는 파라미터 없이 호출되며, 탐색 순서는 **`st.secrets` > 환경변수(`GEMINI_API_KEY`/`GOOGLE_API_KEY`, `.env` 포함)** 두 단계뿐이다. 사이드바 입력 칸을 다시 추가하지 말 것 — 되돌리려면 먼저 사용자에게 확인.
- **`.env` 로딩**: 앱 시작 시 `python-dotenv`의 `load_dotenv()`로 `.env`의 `GEMINI_API_KEY`를 `os.environ`에 적용한다(같은 패턴으로 `python-dotenv` 미설치 시에도 죽지 않게 try/except 처리). 이는 위 우선순위의 "환경변수" 단계를 채우는 로컬 전용 경로다. **`.env`는 절대 커밋하지 않는다**(`.gitignore`에 명시됨). 키 예시 형식은 `.env.example`에 둔다. **Streamlit Cloud 배포 시에는 `.env`가 서버에 존재하지 않으므로, 앱 대시보드의 Secrets에 `GEMINI_API_KEY`를 설정해야 `st.secrets` 경로로 키가 공급된다.**
- **한국어 유지**: 주석·UI 문자열은 한국어로.
- `st.chat_input`은 컬럼 밖(앱 루트 최하단)에 둔다 — Streamlit 배치 제약(컬럼 안에 넣어도 항상 앱 맨 아래 전체 폭에 고정됨) 때문. 컬럼 안으로 옮기지 말 것. 대신 `inject_custom_css()`에서 `[data-testid="stBottomBlockContainer"]`를 `width: calc(50% - 1rem)` + `margin-left: auto`로 강제해 시각적으로 우측(AI 튜터) 컬럼 폭에 맞춘다. **이 testid는 Streamlit 내부 구현이라 버전이 바뀌면 깨질 수 있음** — 업그레이드 후 채팅 입력창이 다시 전체 폭으로 펼쳐지면 이 선택자부터 확인할 것.
- **결정 텍스처(조직) 시각화**: `render_grain_texture_figure()`가 SiO2·깊이로부터 matplotlib Figure를 절차적으로 생성한다(`VISUAL_AVAILABLE` 플래그로 numpy/matplotlib 누락 시 우아하게 degrade — 그림 없이 텍스트만 표시). **그림의 색·알갱이 크기는 연속 보간**이지만 **텍스트 분류 라벨(세립질/반상질/조립질, 현무암질/유문암질 등)은 §2의 단계적 임계값을 그대로 따른다** — 이 함수가 분류 자체를 바꾸지 않도록 주의. 알갱이 생성에는 `(sio2, depth_km)` 기반 고정 시드를 써서 재실행 시 그림이 깜빡이지 않게 한다.

## 5. 알려진 이슈 — "고치지 말 것"(의도된 상태)

> 아래는 버그가 아니라 **사용자가 알고 둔 결정**이다. 임의로 "수정"하지 말고, 손대려면 먼저 확인한다.

- **`google.generativeai` 패키지 지원 종료됨.** 그래도 사용자가 **명시적으로 이 라이브러리를 요청**했다. 마음대로 `google-genai`로 갈아엎지 말 것. 이전이 필요하면 별도 버전으로 제안하고 사용자 승인을 받는다.
- **`gemini-1.5-flash/pro` EOL 가능성**(2026 중반 기준). 모델명은 **사이드바 selectbox에서 교체 가능**하도록 이미 설계됨(`SELECTABLE_MODELS`에 2.0/2.5 포함). 특정 모델로 하드코딩 강제하지 말 것.
- 위 두 가지로 인한 **FutureWarning은 정상**이다. 경고를 없애려고 동작을 바꾸지 말 것.

## 7. 개발/실행 명령
